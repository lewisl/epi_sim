## Running tests

- Build/run all tests:
  - `xmake run test`

- Build/run all tests with artifacts:
  - `xmake run test --artifacts`

- Run one test group:
  - `xmake run test <group>`

- Run one test group with artifacts:
  - `xmake run test <group> --artifacts`
